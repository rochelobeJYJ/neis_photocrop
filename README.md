# 사진 출석부 크롭 도구 (Photo Crop App)

사진 출석부 PDF(나이스 양식 등)에서 학생들 각각의 사진을 인식하여 크롭(잘라내기)하고, 학년/반/번호/이름 정보를 추출하여 자동으로 파일명을 부여해주는 도구입니다.

## 주요 기능
- **다양한 종류의 PDF 지원:** 저화질 스캔본 및 고화질 벡터 PDF(나이스 사진명렬표 등) 두루 지원
- **사진 테두리 및 여백 자동 제거:** 윤곽선/이진화 분석을 통해 불필요한 테두리를 버리고 사진 중심만 크롭
- **자동 파일명 설정:** 상자 안 텍스트나 페이지 헤더에서 추출된 정보로 자유로운 포맷(`{학년}_{반}_{번호}_{이름}.png`)의 파일명 자동 생성
- **간편한 GUI 방식:** 100% 한글, 누구나 쉽게 사용할 수 있는 Windows UI 환경 적용
- **미확인 사진 별도 분류:** 정보 추출에 실패했더라도 안전하게 '미확인_사진'으로 따로 저장하여 유실 방지

## 실행 방법
`photo_crop.zip` 파일을 다운로드 받은 후 압축을 풀고 진행합니다.
파이썬을 설치하지 않아도 실행할 수 있도록 패키징되어 있습니다.

1. 압축을 해제한 폴더로 들어갑니다.
2. `photo_crop_app.exe` 파일을 더블클릭하여 실행합니다.

## 사용 방법
1. **PDF 파일 선택:** 분석할 대상인 사진명렬표 PDF 파일을 엽니다.
2. **저장 폴더 지정:** 크롭된 개별 학생 사진들이 저장될 폴더를 고릅니다.
3. **파일명 변환 포맷 설정:** (기본값) `{학년}_{반}_{번호}_{이름}`과 같은 형태로 저장될 양식을 설정합니다. (정보가 없는 경우 미확인 파일로 저장됩니다. PDF에 있는 정보만 저장될 양식으로 활용하세요.)
4. **미세 조정 (선택 사항):** 상하좌우를 안쪽으로 더 깎고 싶다면 픽셀(px) 단위로 숫자를 입력합니다.
5. **작업 실행:** "▶ 작업 실행" 버튼을 누르고 진행률 바가 끝날 때까지 기다립니다.

## 개발 환경 
- Python 3.10+
- PyMuPDF (fitz)
- OpenCV-Python
- Pillow
- Tkinter

## 빌드 정보
- Windows 환경에서 PyInstaller의 `--onedir` 옵션으로 빌드되었습니다.
